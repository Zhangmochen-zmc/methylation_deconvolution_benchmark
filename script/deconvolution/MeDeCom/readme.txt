# Workflow

The execution is divided into two main steps. Please follow them in order:

## Step 1: Deconvolution (`decon.R`)

```bash
Rscript decon.R
```
## Step 2: Deconvolution process(`decon.R`)

```bash
Rscript decon_process.R
```
We provide pre-processed data files in the test_data that can be used directly for testing the pipeline.
